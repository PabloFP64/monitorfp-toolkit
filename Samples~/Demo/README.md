# Demo Sample

Use this folder for a minimal Unity scene that demonstrates the package.

Recommended demo setup:

- `MonitorServer` with `MjpegTelemetryServer`
- `SessionRecorder` with `SessionRecorder`
- `UserTracker` with `UserTracker`
- a player or XR rig tagged as `Player`
- an interactable object to verify session event logging

If you want, you can later add a small sample scene here without touching the reusable package code.
