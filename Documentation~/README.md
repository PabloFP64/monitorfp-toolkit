# MonitorFP Toolkit Documentation

Package documentation for the reusable Unity toolkit.

## Included features

- MJPEG telemetry server
- Session recording and event logging
- User tracking snapshots
- Top-down map capture helper

## Quick usage

1. Install the package from Git or local path.
2. Add `MjpegTelemetryServer`, `SessionRecorder`, and `UserTracker` to your scene.
3. Use `MonitorFP/Crear Camara Top Down para Mapa` from the Unity menu to create a map camera.
4. Configure `mapRotationDegrees` in `MjpegTelemetryServer` if the map orientation needs adjustment.
